# Responsive SaaS Landing Page

A modern and fully responsive SaaS landing page built using HTML, CSS, and JavaScript as Task 01 of the CovalentX Technologies Web Development Internship.

## Project Overview

SafeFlow AI is a fictional SaaS product landing page designed to showcase a clean, modern, and mobile-friendly user interface for an AI-powered workflow automation platform.

## Features

- Fully responsive design
- Sticky navigation bar
- Navbar color change on scroll
- Mobile menu toggle
- Modern hero section
- SaaS dashboard mockup
- Feature cards
- Pricing section
- Testimonials section
- Call-to-action section
- Clean typography and layout

## Technologies Used

- HTML5
- CSS3
- JavaScript
