# Interactive Navigation Menu

Task 03 of the CovalentX Technologies Web Development Internship.

## Overview

This project demonstrates a modern and interactive navigation system built using HTML, CSS, and JavaScript.

The website showcases various navigation-related user interface interactions including scroll effects, active link highlighting, smooth scrolling, mobile navigation, reveal animations, and a dynamic reading progress indicator.

## Features

### Navigation Features

- Fixed Navigation Bar
- Scroll-Based Navbar Styling
- Active Section Highlighting
- Smooth Scrolling Navigation
- Mobile Hamburger Menu
- Responsive Navigation Layout

### User Experience Features

- Reading Progress Bar
- Back To Top Button
- Scroll Reveal Animations
- Interactive Hover Effects
- Modern Responsive Design

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
